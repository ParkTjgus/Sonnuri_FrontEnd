# 손누리 (Sonnuri) - 프론트엔드

👋 **손누리**는 AI 기반 한국 수어 학습 웹 플랫폼입니다.  
사용자는 수어 영상을 보고 따라하며, 웹캠을 통해 본인의 수어 동작을 실시간으로 인식하고 피드백을 받을 수 있습니다.

## 🌐 프로젝트 개요

- 수어 단어 및 문장 학습
  - 실시간 수어 인식 및 정확도 피드백
- 수어 번역 기능 (한국 수어 ↔ 영어 수어)

## 🛠️ 기술 스택

- **React** + **TypeScript**
- **Vite**
